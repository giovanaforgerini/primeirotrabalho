Obtenha um repositório

Crie uma cópia de trabalho em um repositório local executando o comando git clone /caminho/para/o/repositório.

Quando usar um servidor remoto, seu comando será git clone usuário@servidor:/caminho/para/o/repositório.

Fluxo de trabalho

Seus repositórios locais consistem em três "árvores" mantidas pelo git. a primeira delas é sua Working Directory,que contém os arquivos vigentes.A segunda Index que funciona como uma área temporária e finalmente a HEAD, que aponta para o último commit (confirmação) que você fez.
